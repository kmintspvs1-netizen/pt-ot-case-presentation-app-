# Final Presentation Abstract Template

Use this template when the user wants an abstract for a final, reassessment, or outcome-focused case presentation.

Keep it concise and presentation-ready.
If no target word count is given, aim for a short structured abstract.

## Recommended Structure

### Title

- Diagnosis and main functional outcome or treatment focus

### Introduction

- Why this case and its outcome are worth reporting

### Case Information

- Age, sex, diagnosis, and key baseline status

### Intervention

- Main treatments delivered
- Treatment period and progression

### Results

- Main reassessment findings
- Functional changes
- Goal attainment

### Discussion

- Likely reasons for improvement or limited progress
- Clinical implications

### Conclusion

- Short summary of the key takeaway

## Writing Rules

- Focus on change over time.
- Keep interpretation aligned with the reported results.
- State limitations briefly when they affect the meaning of the outcome.
