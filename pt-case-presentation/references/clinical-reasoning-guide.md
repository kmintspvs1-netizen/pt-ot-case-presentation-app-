# Clinical Reasoning Guide

Use these prompts when the user wants interpretation, discussion, or final consideration.

## Initial Stage Interpretation

- Which finding appears most directly linked to the patient's main activity limitation?
- Which impairments are primary and which are secondary?
- Which findings support the treatment priority?
- What precaution changes the plan or progression?

## Treatment Planning Prompts

- Why is this intervention necessary for the stated goal?
- What change should appear if the intervention is working?
- What should be reassessed to decide whether to progress, continue, or change the plan?

## Reassessment Prompts

- Which outcome improved clearly?
- Which outcome changed only slightly or not at all?
- Did the functional change match the objective change?
- What mismatch remains between impairment-level change and participation-level outcome?

## Discussion Prompts

- What is the most plausible reason for improvement?
- What alternative explanations should be acknowledged?
- Which barriers limited the treatment effect?
- Was the original problem hypothesis appropriate?
- What would be adjusted if the case continued?

## Writing Rules

- State facts first, then interpretation.
- Avoid overclaiming causality when multiple factors could explain change.
- Connect discussion to patient function, not only numeric measures.
- If evidence is weak because data is sparse, say so directly.
