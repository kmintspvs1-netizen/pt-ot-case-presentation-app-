# Initial Evaluation Abstract Template

Use this template when the user wants an abstract for an initial evaluation or early-stage case presentation.

Keep it concise and presentation-ready.
If no target word count is given, aim for a short structured abstract.

## Recommended Structure

### Title

- Diagnosis and main functional issue

### Introduction

- Why this case is clinically important

### Case Information

- Age, sex, diagnosis, living situation, and key background

### Evaluation Findings

- Main impairments
- Activity limitations
- Participation restrictions

### Clinical Interpretation

- Main problem hypothesis
- Why the selected treatment direction is appropriate

### Treatment Plan

- Main interventions planned
- Functional aim of treatment

### Conclusion

- Short summary of the case significance and expected direction

## Writing Rules

- Keep the abstract tighter than the full presentation.
- Prefer one clear point per sentence.
- Avoid over-detailed exercise prescriptions unless required by the target format.
