# 画像生成プロンプト雛形

患者向け自主トレの説明画像を、Gemini の nanobanana や ChatGPT などで手動生成するときのテンプレート。
毎回すべてを自由記述にせず、必要項目を差し替えて使う。

## このテンプレートの位置づけ

- このスキルでは、画像生成サービスへの送信自体は自動化しない。
- Codex はプロンプト作成、HTML 作成、生成後画像の差し替えを担当する。
- ユーザーは Gemini や ChatGPT にこのプロンプトを貼り、画像を手動生成する。

## 基本テンプレート

```text
患者向け自主トレ説明用のイラストを作成してください。
目的は「{運動名}」をわかりやすく説明することです。

条件:
- 対象: {年齢層}、{疾患・状態}
- 開始姿勢: {開始姿勢}
- 動作: {動作の説明}
- 強調したい部位: {部位}
- 支えの有無: {手すり・椅子・壁など}
- 画像の見た目: シンプル、清潔感、医療説明向け、背景は最小限
- 構図: {全身 / 上半身 / 下肢中心 / 横向き / 正面 など}
- 補助表現: 動く方向に矢印、必要なら左右ラベル
- 避けること: 過度に写実的、複雑な背景、不要な小物、誤解を招く姿勢
- 出力目的: A4 の患者説明用紙に載せるため、ひと目で動作が伝わることを最優先
```

## 短めテンプレート

急いでいるときは、以下の短縮版でもよい。

```text
患者向け自主トレ用の説明イラストを作成してください。
運動名: {運動名}
対象: {年齢層}、{疾患・状態}
開始姿勢: {開始姿勢}
動作: {動作}
強調部位: {部位}
支え: {あり / なし}
見た目: シンプル、背景最小限、医療説明向け
構図: {構図}
追加: 矢印と左右ラベルを必要に応じて入れる
避けること: 誤った姿勢、複雑な背景、不要な装飾
```

## 記入例 1

```text
患者向け自主トレ説明用のイラストを作成してください。
目的は「椅子に座って行う膝伸ばし運動」をわかりやすく説明することです。

条件:
- 対象: 高齢者、変形性膝関節症
- 開始姿勢: 椅子に浅く座る
- 動作: 片脚の膝をゆっくり伸ばし、元に戻す
- 強調したい部位: 膝関節と下腿の動き
- 支えの有無: 椅子あり
- 画像の見た目: シンプル、清潔感、患者説明向けイラスト
- 構図: 横から見た全身
- 補助表現: 下腿が前に伸びる方向へ矢印を入れる
- 避けること: 複雑な背景、筋肉誇張、若すぎるモデル、誤った姿勢
- 出力目的: A4 の自主トレ説明用紙に載せる
```

## 記入例 2

```text
患者向け自主トレ説明用のイラストを作成してください。
目的は「壁に手をついて行うかかと上げ運動」をわかりやすく説明することです。

条件:
- 対象: 高齢者、バランス低下
- 開始姿勢: 壁に両手をついて立つ
- 動作: 両かかとをゆっくり持ち上げ、ゆっくり下ろす
- 強調したい部位: 足首とかかとの上下運動
- 支えの有無: 壁あり
- 画像の見た目: シンプル、清潔感、リハビリ説明向け
- 構図: 横から見た全身
- 補助表現: かかとが上がる方向に矢印
- 避けること: 片脚立ちに見える表現、不安定な姿勢、雑多な背景
- 出力目的: 自宅練習用 handout に載せる
```

## 使い分けのコツ

- 膝、股関節、足関節の運動は横向き構図だと動きが伝わりやすい。
- 体幹や上肢の左右差を見せたいときは正面構図も有効。
- 立位課題では、支えがあることを画像内で明確に示す。
- 1 枚に複数動作を入れるより、1 動作 1 画像のほうが患者向けには伝わりやすい。
- 生成結果に誤った姿勢が含まれたら、角度、支持物、足の位置をより具体的に再指定する。
