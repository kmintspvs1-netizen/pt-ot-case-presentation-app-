# note販売ページ用紹介文

## タイトル案

【理学療法士が設計】運動が「歯磨き」になる90日間ロードマップ

## 短めの紹介文

運動が続かないのは、意志が弱いからではありません。

理学療法士として見てきた「続く人」と「続かない人」の違いをもとに、90日で運動を習慣化するための考え方と行動設計を1本にまとめました。個別サポートなし、追加課金なし。読んで、そのまま生活に落とし込める記事完結型のプログラムです。

## 販売ページ向けの説明文

何度も運動に挫折してきた人へ。

ジム入会、宅トレ、散歩、ストレッチ。始めては止まり、止まってはまた自己嫌悪になる。その繰り返しに疲れていないでしょうか。

このnoteは、そんな人のために作りました。

内容は、気合いや根性の話ではありません。理学療法士として臨床で見てきた「続く人の共通点」と、行動科学・習慣化の考え方をもとに、90日で運動を生活に定着させるためのロードマップを整理しています。

この記事で扱うのは、次のような内容です。

- 運動が続かない人に共通する思考のクセ
- 「やる気頼み」を卒業するための設計
- 5分から始める具体的な習慣化のコツ
- 忙しい日、疲れた日、雨の日の例外ルール
- 2か月目の飽き、3か月目の停滞への対処
- 年代別、デスクワーカー向け、膝腰に不安がある人向けの調整ポイント

このnoteは、個別サポートへ誘導するための入口記事ではありません。

買って読んで、自分で使えることを重視して作っています。だからこそ、派手な煽りよりも、現実に続けやすいことを優先しています。

「運動が好きになれない」
「時間がない」
「何度も失敗してきた」

そんな人ほど、相性のいい内容です。

## こんな人におすすめ

- 運動を始めてもいつも3日坊主になる
- 忙しくて長い運動時間は取りにくい
- 自分を責めずに習慣化したい
- 一人で静かに整えていきたい
- 身体のことを分かった人の視点で学びたい

## こんな人には向かないかもしれません

- すでに運動習慣が安定している
- 高強度トレーニングのメニューだけ知りたい
- 短期間で大きな減量だけを狙っている
- 個別指導や伴走サポートを求めている

## 価格説明に使える文

価格は1,480円を想定しています。

ジム1か月分よりずっと低い負担で、「何をやるか」だけでなく「どうすれば続くか」までまとめて確認できる内容です。単発のやる気ではなく、90日で生活の中に定着させるための設計図として読んでいただければと思います。

## 告知文の短文案

運動が続かないのは、意志が弱いからではなく、設計が合っていないだけかもしれません。
