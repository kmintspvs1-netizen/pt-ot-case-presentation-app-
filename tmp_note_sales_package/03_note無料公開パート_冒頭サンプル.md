# note無料公開パート用サンプル

「運動しないといけないのは分かっている。でも続かない」

この悩みは、意志が弱い人だけのものではありません。むしろ、多くの人が同じ場所でつまずいています。

ジムに入会したけれど数か月で行かなくなった。動画を見ながら家で筋トレを始めたけれど、忙しくなって止まった。気合いを入れて再開したのに、疲れたり筋肉痛がきつかったりして、また遠のいた。

こうした流れを何度も経験すると、人は少しずつ「自分は続けられない人間なんだ」と思い始めます。でも、それは違います。

運動が続かないのは、根性がないからではありません。多くの場合は、始め方と続け方の設計が合っていないだけです。

私は理学療法士として、臨床でたくさんの人の「続く」と「続かない」を見てきました。リハビリも、本質的には運動の継続です。そこで感じたのは、続く人は特別にストイックなのではなく、考え方と行動の組み立てが上手いということでした。

この文章では、90日かけて運動を「がんばるもの」から「やらないと落ち着かないもの」へ変えていく流れを、順番どおりにまとめています。

個別サポートはありません。ですが、その代わりに、ひとりで読み進めても実践しやすいように、考え方、ワーク、例外ルール、続け方のコツをひとつの流れに整理しました。

対象は、こんな人です。

- これまで何度も運動に挫折してきた
- 忙しくてまとまった時間が取りにくい
- 体力にあまり自信がない
- 意志ではなく、仕組みで続けたい
- できれば一人で静かに整えていきたい

逆に、すでに週4回以上の運動が無理なく習慣化できている人には、この内容はそこまで必要ないかもしれません。

このプログラムで目指すのは、90日後に「今日はやる気があるから運動する」ではなく、「いつもの流れとして体を動かす」に変わっている状態です。

大事なのは、最初から完璧を目指さないことです。この記事全体を通して、繰り返しお伝えしたいことがあります。

それは、運動習慣は気合いではなく設計で作れる、ということです。
