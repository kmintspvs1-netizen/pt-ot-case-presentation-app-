# 脊柱管狭窄症術後 自主トレ画像プロンプト

以下は、今回の A4 用紙に載せる 4 種目を Gemini で手動生成するときの実用寄りプロンプト。
患者さんが図を見て真似しやすいことを最優先にしている。

## 使い方

- まず 1 種目ずつ生成する。
- 「患者向けの説明図」「高齢男性」「リハビリ指導用」「背景は白か薄い単色」を毎回入れる。
- 1回目でうまくいかないときは、`もっと横向き` `脚の角度を自然に` `壁に手をついている姿勢を明確に` のように修正指示を足す。
- 矢印や文字ラベルは Gemini に描かせてもよいが、後から別で入れる前提でもよい。

## 1. おしり上げ

```text
高齢男性の患者向けリハビリ説明図を作成してください。
脊柱管狭窄症術後約2か月の方に配る自主トレ用紙に載せるイラストです。

描いてほしい内容:
- 運動名: おしり上げ
- 姿勢: ベッドまたは布団の上で、あお向け、両ひざを立てる
- 動作: おしりを少しだけ持ち上げる
- 構図: 横から見た全身
- 見た目: 医療説明用、やさしい雰囲気、清潔感、背景は白またはごく薄い単色
- 強調点: 骨盤が少し上がること、腰を反らせすぎていないこと
- 補助: 持ち上げる方向に小さな矢印

避けてほしいこと:
- 若すぎる人物
- 筋肉質すぎる人物
- おしりを高く上げすぎたブリッジ
- 複雑な家具や生活背景
- ヨガやフィットネスのような派手な雰囲気
```

## 2. ひざ抱え

```text
高齢男性の患者向けリハビリ説明図を作成してください。
脊柱管狭窄症術後約2か月の方に配る自主トレ用紙に載せるイラストです。

描いてほしい内容:
- 運動名: 片ひざ抱え
- 姿勢: ベッドまたは布団の上であお向け
- 動作: 片ひざを両手で軽く抱えて、胸に近づける
- 構図: 横から見た全身
- 見た目: 医療説明用、やさしい雰囲気、背景は白、人物が見やすい
- 強調点: ひざを軽く引き寄せる動き、無理のない範囲
- 補助: ひざが胸に近づく方向に矢印

避けてほしいこと:
- 強く引き寄せすぎた姿勢
- 首や肩に力が入りすぎている姿勢
- 若いスポーツ選手のような体つき
- 複雑な背景やベッド周りの小物
```

## 3. 横向き脚上げ

```text
高齢男性の患者向けリハビリ説明図を作成してください。
脊柱管狭窄症術後約2か月で、右股関節外転筋が弱い方に配る自主トレ用紙に載せるイラストです。

描いてほしい内容:
- 運動名: 横向き脚上げ
- 姿勢: 横向きで寝る、下の脚は軽く曲げる、上の脚は軽く伸ばす
- 動作: 上の脚を少しだけ持ち上げる
- 構図: 横から見た全身
- 見た目: 医療説明用、シンプル、背景は白
- 強調点: 体が後ろに倒れていないこと、脚を高く上げすぎないこと
- 補助: 上の脚が少し上がる方向に矢印

避けてほしいこと:
- バレエのように脚を高く上げた姿勢
- 骨盤が後ろに倒れた姿勢
- 若く見えるモデル
- 装飾の多い背景
```

## 4. 前かがみストレッチ

```text
高齢男性の患者向けリハビリ説明図を作成してください。
脊柱管狭窄症術後約2か月で、杖歩行、転倒リスクがある方に配る自主トレ用紙に載せるイラストです。

描いてほしい内容:
- 運動名: 前かがみストレッチ
- 姿勢: 椅子または安定した机の前に座り、両手を机または太ももに軽く置く
- 動作: 背中を丸めすぎずに、体を少し前へ倒す
- 構図: 横から見た全身
- 見た目: 医療説明用、背景は白、支えが明確に見える
- 強調点: 無理のない前屈、腰やおしりまわりが軽く伸びる姿勢
- 補助: 体が前に倒れる方向に小さな矢印

避けてほしいこと:
- 立位姿勢になってしまうこと
- 深くかがみすぎる姿勢
- 背中を極端に丸めた姿勢
- 背景が複雑な室内
- フィットネスやヨガのような見た目
```
