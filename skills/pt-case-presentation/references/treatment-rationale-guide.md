# Treatment Rationale Guide

Use this guide when the user wants the current treatment summarized, justified, or supported with references.

## 1. Confirm Current Treatment

Identify what is already being done before proposing additions.

Examples:

- standing balance practice barefoot
- ergometer
- gait practice with walker
- sit-to-stand practice
- ROM exercise
- strengthening exercise

If the current treatment is unknown, ask briefly or write `current treatment not provided`.

## 2. Link Each Treatment to the Case

For each treatment item, explain:

- what problem it targets
- what functional goal it supports
- what sign would suggest it is helping
- what precaution may limit intensity or progression

## 3. Search and Summarize Rationale

When evidence is requested:

- prefer clinical practice guidelines, systematic reviews, randomized trials, and other primary sources
- summarize evidence briefly
- avoid long quotations
- state when the evidence supports the general approach rather than the exact protocol being used

## 4. Recommended Output Format

### Current Treatment

- treatment item
- short description of how it is being used in this case

### Current Treatment and Rationale

- treatment item
- target problem
- brief rationale
- precaution

### Sources

- list the source title and link

## 5. Example Framing

- `Barefoot standing balance practice targets low balance ability and reduced sensory confidence during stance. In this case it may support safer standing control and gait preparation.`
- `Ergometer can help maintain or improve aerobic tolerance while reducing direct walking load. In this case it may be useful when walking endurance is limited but activity needs to be maintained.`

## 6. Caution

- Do not present a rationale as definitive proof.
- Separate published evidence from case-specific interpretation.
- Highlight precautions such as DVT, post-operative restriction, pain flare, orthosis use, or physician instructions.
