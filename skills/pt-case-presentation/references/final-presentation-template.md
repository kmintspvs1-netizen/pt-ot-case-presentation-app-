# Final Presentation Template

Use this template when the user wants a post-treatment, reassessment, or discharge-oriented presentation.

## 1. Case Overview

- Diagnosis
- Onset or surgery timing
- Initial functional status
- Main goals at the start of intervention

## 2. Initial Evaluation Summary

- Key baseline impairments
- Activity and participation limits
- Main clinical interpretation from the initial stage

## 3. Intervention Summary

- Treatment period and frequency if provided
- Major interventions performed
- Rationale for the overall progression

## 4. Clinical Course

- Important changes during the intervention period
- Response to treatment
- Any setbacks, pain flare, or barriers

## 5. Reassessment Findings

- Pain
- Range of motion
- Strength
- Balance
- ADL or gait
- Standardized outcomes if available
- Comparison with baseline

## 6. Results

- Goal attainment
- Functional improvement
- Residual problems

## 7. Discussion

- What likely contributed to improvement
- What limited progress
- Whether the intervention matched the main problem hypothesis
- How the findings connect to the final functional outcome

## 8. Future Issues

- Remaining risks
- Next-stage treatment needs
- Home exercise, self-management, or community reintegration points

## 9. Closing Summary

- One-paragraph final presentation summary
- Optional speaker script if requested
